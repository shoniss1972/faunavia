# Faunavia

**Faunavia** is a mobile-first wildlife rescue and transport game concept combining short driving challenges, route planning, vehicle upgrades, and light animal-management puzzles.

The player begins with a bicycle and rear cage, then progresses through motorcycles, flat decks, jeeps, trucks, trailers, and specialised transport modules. Missions require the player to select suitable equipment, collect expressive animals, manage fuel and supplies, navigate terrain, and deliver every passenger comfortably.

## Current status

Early prototype. The immediate goal is to prove that transporting opinionated animals creates a satisfying, replayable mobile gameplay loop.

## Prototype principles

- Puzzle first; simulation second.
- Short, self-contained mobile levels.
- No visible injury, suffering, predation, or death.
- Negative outcomes use humour, inconvenience, refusal, and animal disapproval.
- No multiplayer, advertising, purchases, accounts, or live-service systems during prototyping.
- Build the smallest playable loop before adding architecture or content pipelines.

## Technology

- Godot standard edition
- GDScript
- Portrait mobile layout

## Run locally

1. Install the current stable standard edition of Godot.
2. Clone this repository.
3. Import `project.godot` into Godot.
4. Run the project.
